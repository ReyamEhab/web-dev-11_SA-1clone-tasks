What is HTML?
(HYPER TEXT MARKUP LANGUAGE) The set of markup symbols or codes placed in a file intended for display on a Web browser page.

  • Head Section

    head Contains information that describes the web page document

  • Body section
    •       <body>body section</body>

    body Contains text and elements that display in the web page document

Div element

      <div><The text goes here><div>

div Configures a structural block area or “division” on a web page with empty space above and below.
